
# Exit Poll Centuripe – Web App (Starter)

Questo è uno **starter project** per la web app degli exit poll.

## Struttura
- Next.js frontend
- Pagine tablet
- Pagina admin
- Schema SQL per Supabase

## Avvio locale
1. installa Node.js
2. nel terminale:

npm install
npm run dev

Poi apri:
http://localhost:3000

## Deploy consigliato
- Vercel
- Database Supabase
- Dominio: app.exitpollcenturipe.it
